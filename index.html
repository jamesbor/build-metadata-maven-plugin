Build Meta Data Maven Plugin

  • This plugin is intended to get information about the current build environment and provide it.

Goals Overview

  • build-helper:maven-version Set a property containing the current version of maven.
  • build-helper:local-hostname Retrieve current hostname.
  • build-helper:java Retrieve information about Java environment.
  • build-helper:os Retrieve information about current operation system.
  • build-helper:username Retrieve information about currently logged in user.

Usage
