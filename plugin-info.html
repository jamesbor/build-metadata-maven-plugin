Plugin Documentation

Goals available for this plugin:

GoalReport?Description
build-metadata:helpNoDisplay help information on build-metadata-maven-plugin.
Call mvn build-metadata:help -Ddetail=true -Dgoal=<goal-name> to display parameter details.
build-metadata:hostnameNoRetrieve current hostname and place it under a configurable project property
build-metadata:javaNoSet the java properties which contain the information about java runtime, java specification etc. which has been used during the build. 
  [propertyPrefix].java.runtime.name
  [propertyPrefix].java.runtime.version
  [propertyPrefix].java.specification.name
  [propertyPrefix].java.specification.vendor
  [propertyPrefix].java.specification.version
  [propertyPrefix].java.version
  [propertyPrefix].java.vendor
  [propertyPrefix].java.vm.name
  [propertyPrefix].java.vm.info
  [propertyPrefix].java.vm.version
  [propertyPrefix].java.vm.vendor
  [propertyPrefix].java.vm.specification.name
  [propertyPrefix].java.vm.specification.vendor
  [propertyPrefix].java.vm.specification.version
  [propertyPrefix].sun.management.compiler
Where the propertyPrefix is the string set in the mojo parameter.
build-metadata:mavenNoGet maven-metadata which means the version of Maven the active profiles. 
  [propertyPrefix].version
  [propertyPrefix].profiles
  [propertyPrefix].goals
build-metadata:metadataNoRetrieve the metadata Java, Java OPTS, operation system, maven information and provide it as properties accessible in the project.
build-metadata:metadatareportYesRetrieve current username and place it under a configurable project property
build-metadata:osNoSet the os properties which contain name, family, version and architecture of the operation system. 
  [propertyPrefix].name
  [propertyPrefix].family
  [propertyPrefix].version
  [propertyPrefix].arch
Where the propertyPrefix is the string set in the mojo parameter.
build-metadata:usernameNoRetrieve current username and place it under a configurable project property

System Requirements

The following specifies the minimum requirements to run this Maven plugin:

Maven2.2.1
JDK1.5
MemoryNo minimum requirement.
Disk SpaceNo minimum requirement.

Usage

You should specify the version in your project's plugin configuration:

<project>
  ...
  <build>
    <!-- To define the plugin version in your parent POM -->
    <pluginManagement>
      <plugins>
        <plugin>
          <groupId>com.soebes.maven.plugins</groupId>
          <artifactId>build-metadata-maven-plugin</artifactId>
          <version>1.0-SNAPSHOT</version>
        </plugin>
        ...
      </plugins>
    </pluginManagement>
    <!-- To use the plugin goals in your POM or parent POM -->
    <plugins>
      <plugin>
        <groupId>com.soebes.maven.plugins</groupId>
        <artifactId>build-metadata-maven-plugin</artifactId>
        <version>1.0-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </build>
  ...
  <!-- To use the report goals in your POM or parent POM -->
  <reporting>
    <plugins>
      <plugin>
        <groupId>com.soebes.maven.plugins</groupId>
        <artifactId>build-metadata-maven-plugin</artifactId>
        <version>1.0-SNAPSHOT</version>
      </plugin>
      ...
    </plugins>
  </reporting>
  ...
</project>

For more information, see "Guide to Configuring Plug-ins"
