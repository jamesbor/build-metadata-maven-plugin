build-metadata:metadatareport

Note:This goal should be used as a Maven report.

Full name:

com.soebes.maven.plugins:build-metadata-maven-plugin:1.0-SNAPSHOT:metadatareport

Description:

Retrieve current username and place it under a configurable project property

Attributes:

  • Requires a Maven project to be executed.
  • Executes only as a reportSet (reporting goal).
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: site.

Parameters

(no parameters)
