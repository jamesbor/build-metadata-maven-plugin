build-metadata:hostname

Full name:

com.soebes.maven.plugins:build-metadata-maven-plugin:1.0-SNAPSHOT:hostname

Description:

Retrieve current hostname and place it under a configurable project property

Attributes:

  • Requires a Maven project to be executed.
  • The goal is thread-safe and supports parallel builds.
  • Binds by default to the lifecycle phase: validate.

Optional Parameters

Name Type Since Description
defaultPropertyValue String - The default value for undefined properties.
propertyPrefix String - The name of the property in which to store the local hostname.
Default value is: build.

Parameter Details

defaultPropertyValue:

The default value for undefined properties.
  • Type: java.lang.String
  • Required: No

propertyPrefix:

The name of the property in which to store the local hostname.
  • Type: java.lang.String
  • Required: No
  • Default: build
